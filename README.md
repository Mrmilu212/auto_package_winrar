# auto_package_winrar

一个 Windows 图形界面小工具：把**文件/文件夹**拖放到窗口中，自动调用 **WinRAR (Rar.exe)** 进行打包，并支持二次/三次（分卷）压缩。

## 功能

- **拖放打包**：拖放文件或文件夹到窗口即可开始压缩
- **一次压缩**：压缩包内容与拖放对象一致  
  - 拖放文件夹：压缩包里包含该**文件夹本身**  
  - 拖放文件：压缩包里只包含该**文件本身**
- **二次压缩**：将第一次压缩产物改为随机“伪装后缀”（如 `.png`、`.dll`、`.mp4` 等）后，再对该伪装文件进行打包
- **三次压缩（分卷）**：在二次压缩基础上继续打包，并按规则分卷
  - **至少 2 卷**
  - **每卷最大 2GB**
- **随机命名压缩包**：每进行一次压缩都会生成 **5 位**随机文件名（`A-Za-z0-9`）
  - 三次分卷最终输出形如：`ABCDE.part1.rar`、`ABCDE.part2.rar` ...
- **已用时间显示**：状态栏实时显示已用时间，并显示当前在进行第几次压缩（不显示百分比与剩余时间）
- **取消**：压缩过程中可点击“取消”终止；关闭窗口也会触发取消
- **原子性**：要么整套流程全部成功并输出最终产物，要么失败/取消时不在目标目录留下任何半成品

## 环境要求

- Windows 10/11
- 已安装 **WinRAR**，且存在命令行工具 `Rar.exe`
  - 常见路径：`C:\Program Files\WinRAR\Rar.exe`
- Python 3（推荐 3.10+），并确保能使用 `pythonw.exe`（用于无控制台启动）

## 安装

在项目目录执行：

```powershell
python -m pip install -r requirements.txt
```

## 启动

- **推荐**：双击 `启动.bat`
- 或命令行启动：

```powershell
python main.py
```

## 使用

1. 打开程序窗口
2. 勾选需要的模式：
   - 不勾选：一次压缩
   - 勾选“二次压缩”：二次压缩
   - 勾选“三次压缩（分卷）”：三次压缩（会自动联动二次）
3. 将文件/文件夹拖放到窗口中

输出位置：默认输出到拖放对象所在目录（压缩包文件名为随机 5 位）。

## 常见问题

- **未找到 WinRAR（Rar.exe）**
  - 请确认已安装 WinRAR
  - 或将 `Rar.exe` 所在目录加入系统 PATH

- **双击启动出现提示找不到 pythonw**
  - 请安装 Python，并在安装时勾选 **Add Python to PATH**
  - 或手动把 Python 安装目录加入 PATH（其中应包含 `pythonw.exe`）

## 说明

本工具仅调用 WinRAR 命令行 `Rar.exe` 完成压缩/分卷等工作；界面仅做任务编排、随机命名、原子提交与取消控制。
